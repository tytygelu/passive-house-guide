# I-Tayili Zokhuni Eziphumelelayo (LVT): Umhlahlandlela Ophelele Wokufaka Ukuze Ufinyelele Umphumela Ongaphazamiseki

## Izidingo Eziyinhloko

Ukufakwa kwe-LVT kudinga ukunakwa okucophelela emininingwaneni ngokwe-BS 8203:2017:

- Ukuhlelwa kwesisekelo sokufaka okuphumelelayo
- Ukuphathwa kahle kokunwetshwa
- Ukukhetha imikhiqizo efanele
- Izindlela zokufaka ezisebenzayo
- Izinqubo zokuphetha ezisezingeni

## Inqubo Yokufaka

### Ukuhlelwa Kwasemva
- Ukususwa kwemishini kwe-laitance
- Ukususa izinto ezingcolile ebusweni
- Ukuhlolwa kokuphumelelayo okujulile
- Ukuhlolwa kokuphuma
- Izimiso eziyisisekelo zokuhlela
- Ukuhambisana nezindinganiso zaseBrithani

### Ukuphathwa Kwamanzi
- Ukuhlolwa kokunisela okungukwaziyo
- Ukusetshenziswa kwe-hygrometer elungisiwe
- Ukuhlolwa kwezinga le-RH (okuphezulu 75%)
- Ukufakwa kwememori engangeni manzi:
  - Uhlaka olulodwa lwe-RH uphakathi kuka-98%
  - Isikhathi sokuphilisa sehora ezintathu
  - Ukuhlukaniswa okuphelele kokunisela
  - Ukuvimbela ukuwa kwephezulu

## Izinto Eziyinhloko

### Izixazululo Zokuphakamisa
- Ukulungiselela okubalulekile kwendawo
- Ukusebenza okuhle kwe-compound
- Ukwelashwa kwe-subfloor ethokozisayo
- Izinketho zokonga isikhathi zikhona
- Izicelo ezikhethekile:
  - Ama-primers ajwayelekile
  - Ama-primers angathathi amanzi
  - Ama-primers e-calcium sulphate screed

### Izixazululo Zokulinganisa
- Ukwakhiwa kwendawo efanele
- Ukuqinisekiswa kokubukeka
- Izinketho ezinzima zikhona:
  - Amandla aphezulu okucindezela
  - Ukuzithambisa okuhle
  - Ikhono lokuthwala umthwalo omkhulu
  - Ukubekezelela ukuhamba okukhulu

### Izixazululo Ezinokuguquguquka
- Izicelo ze-plywood subfloor
- Ukufakwa kwezinsimbi
- Ukuvuma ukuhamba
- Izinyathelo zokuvimbela ukuvuka
- Ukuvikelwa kokubukeka

## Izincazelo Zobuchwepheshe

| Isici | Isidingo |
|-------|----------|
| Umkhawulo We-RH | 75% standard, 98% nge-membrane |
| Isikhathi Sokuphilisa | Amahora amathathu ngememori |
| Ukuhlanganiswa | Kuhluka ngomkhiqizo |
| Izindinganiso | Kuhambisana ne-BS 8203:2017 |
| Izinhlobo Zezindawo | Ikhonkolo, plywood, insimbi |
| Ukuhlola Ukuhamba | Izinketho ezinzima zikhona |

## Izindlela Zokufaka

### Ukufaka Okunamathelayo
- Izixazululo ezithintekayo
- Izici zokubamba ngokushesha
- Ikhono lokwakha amaphethini
- Izindlela ezifanele zokusebenzisa i-roller:
  - Ukufaka i-roller yokupenda
  - Ukunciphisa izigaxa
  - Ukuvimbela izimpawu ze-trowel
  - Ukuvikelwa kokubukeka

### Ukuqapha Ikhwalithi
- Ukuqinisekiswa kokuhambisana
- Imihlahlandlela yomkhiqizi
- Ukubonisana ne-RAG
- Ngaphezu kwe-5,000 kwezokumboza phansi
- Ukuvikelwa kwe-200+ komkhiqizi

## Izinselelo Ezivamile

### Izinkinga Zokuphuma Kwamanzi
- Umthombo ophambili wokwehluleka kwephansi
- Ukuphuma kwamanzi okusele kokwakha
- Izinkinga zokuphakama kwamanzi
- Ukwehla kokunamathelayo
- Ukuvuvukala kwesurface
- Ukuphakama kokufaka

### Izinyathelo Zokuvikela
- Ukuhlolwa kokuphuma kwamanzi okufanele
- Ukusetshenziswa kwememori efanele
- Ukukhetha i-primer efanele
- Ukufaka okwakhiwe kahle
- Ukufaka okuhlelekile

## Izindlela Eziphumelelayo

### Izidingo Zobuchwepheshe
- Ukuhambisana nezindinganiso
- Ukukhetha amathuluzi afanele
- Ukufaka umkhiqizo ofanele
- Izinyathelo zokulawula ikhwalithi
- Ukuhlolwa okujwayelekile

### Ukukhetha Izinto
- Ukukhetha i-primer efanele
- Ukukhetha okwakhiwe okufanele
- Ukusetshenziswa kokunamathelayo okuhambisanayo
- Ukuhlola ikhwalithi
- Ukuhlinzekwa komkhiqizi

## Izinzuzo Zesikhathi Eside

### Izinzuzo Zokusebenza
- Impilo yokufaka ebanzi
- Ukubukeka okuqhubekayo
- Ukuqina kwesakhiwo
- Ukumelana nokuhamba
- Ikhono lokuthwala umthwalo

### Ukuze Kwenziwe Ngcono Izindleko
- Ukunciphisa ukugcinwa
- Ukuvimbela ukubuyiselwa
- Ukunciphisa ukulungiswa
- Ukuphila isikhathi eside
- Ukwaneliseka kwamakhasimende
