# 豪华乙烯基地板（LVT）：完美安装指南

## 核心要求

LVT安装要求根据BS 8203:2017进行细致的关注：

- 良好的基层准备
- 适当的湿度管理
- 合适的产品选择
- 专业的安装技术
- 高质量的修饰程序

## 安装过程

### 基层准备
- 机械去除浮浆
- 消除表面污染物
- 彻底的坚固性检查
- 平整度评估
- 基本准备原则
- 符合英国标准

### 湿度管理
- 强制湿度测试
- 使用校准的湿度计
- 相对湿度水平评估（最高75%）
- 防水膜应用：
  - 单层涂布适用于最高98%相对湿度
  - 三小时固化时间
  - 完全湿度隔离
  - 防止地板失效

## 关键组件

### 底漆解决方案
- 必要的表面准备
- 优化的材料性能
- 吸水性基层处理
- 节省时间的选项可用
- 专业应用：
  - 通用底漆
  - 非吸水性表面底漆
  - 硫酸钙找平底漆

### 平整材料
- 完美平整表面创建
- 视觉外观保证
- 重型选项可用：
  - 高抗压强度
  - 优秀的自平整性
  - 重载能力
  - 高交通耐受性

### 灵活解决方案
- 胶合板基层应用
- 钢表面安装
- 运动适应
- 裂缝预防措施
- 外观保持

## 技术规格

| 特征 | 要求 |
|------|------|
| 相对湿度水平限制 | 75% 标准，使用膜时为98% |
| 固化时间 | 膜的固化时间为3小时 |
| 覆盖范围 | 根据产品而异 |
| 标准 | 符合BS 8203:2017 |
| 表面类型 | 混凝土、胶合板、钢材 |
| 交通等级 | 提供重型选项 |
