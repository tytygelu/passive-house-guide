# Breedon Generon Soltak: Integrerad Förnybar Energilösning

## Innovativ Solintegration

Breedon Group har samarbetat med den europeiska takspecialisten Terran för att lansera Generon - ett diskret soltakssystem som kombinerar:

- 3,2 mm monokristallina PV-celler
- Integrerat med Elite 330 mm x 420 mm betongbasplattor
- Sömlös installation med standardmatchande plattor
- 260-plattorssystemet genererar typisk 4 kW effekt

## Nyckelfördelar

✅ **Estetisk Integration**  
Eliminerar skrymmande solpaneler genom:
- Planmonterad härdat glasytan
- Färgmatchade betongsubstrat
- Kontinuerlig takytans utseende

✅ **Förenklad Installation**  
Takentreprenörer kan installera med hjälp av:
- Standard stormklippfästen
- Förinstallerat sammankopplingssystem
- Ingen separat solmonteringsutrustning

✅ **Förbättrad Hållbarhet**  
Testad för att klara av:
- 120 mph vindar
- Basebollstora hagelimpakter
- 50-årig livslängd för betongplattor
- 20-årig prestandagaranti

## Regulatorisk Efterlevnad

Hjälper till att uppfylla del L-kraven genom:
- Förnybar energiproduktion på plats
- Minskad koldioxidavtryck vid drift
- Smart energimonitorering via husägarapp
