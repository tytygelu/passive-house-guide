# 未来の住宅基準2025：屋根と断熱の変革

## コア要件

未来の住宅基準（FHS）は、2025年から建設される新しい住宅に対して厳格な要件を導入します：

- 現在と比較して75-80%の炭素排出削減
- 強化された換気システム
- 改善されたエネルギー効率基準
- ファブリックファーストアプローチへの焦点
- 最適化された熱性能
- 業界リーダーによる積極的な修正
- 包括的な持続可能性アプローチ

## 主要コンポーネント

### 屋根ソリューション
- 伝統的なオプションとしてのコンクリートタイル
- 美的魅力のためのテラコッタタイル
- 市場シェアを獲得しているファイバーセメントスレート
- ソーラーパネル互換性の計画
- 屋根からの熱損失の25%
- 適切な換気が不可欠
- 現在のところソーラーパネルの義務はなし
- 将来の統合に向けた論理的な計画

### ファサード材料
- 木材建設オプション
- 耐久性のある石のファサード
- ビニールクラッディングソリューション
- 金属システムの統合
- ウェザーボードの代替
- ファイバーセメントの利点:
  - 強くて多用途
  - 持続可能な構成
  - 原材料使用量の削減
  - 製造時のエネルギー削減
  - 廃棄物の削減
  - A1火災分類
  - 極端な温度耐性
  - 低メンテナンス要件

### 一般的な組み合わせ
- 一階のレンダリング
- クラッディングを施した上層階（例：セドラル）
- 混合材料アプローチ
- 美的考慮
- パフォーマンスの最適化

## 断熱戦略

### 外部断熱
- レインスクリーンクラッディングシステム
- エネルギー効率の向上
- ファサードの寿命延長
- 結露の減少
- 構造的動きの最小化
- 天候保護の利点
- 熱橋の削減

### 内部断熱
- ミネラルウールロール
- 木製バテンシステム
- 安定した内部気候
- 保存された外観
- コスト効率の良いソリューション
- スペースの考慮が必要
- 防火被覆要件

## 技術的考慮事項

| 特徴 | 仕様 |
|------|------|
| 炭素削減 | 現行基準に対して75-80% |
| 屋根からの熱損失 | 建物全体の熱の25% |
| 繊維セメントの防火等級 | A1分類 |
| 設置オプション | 外部または内部 |
| 換気 | 計画が必須 |
| 持続可能性 | 高い再利用素材含有率 |
| 温度性能 | 極端な耐性 |
| メンテナンス要件 | 最小限 |
