# Breedon Generon Solar Roof Tile: Integrado Energía Renovable Solución

## Innovadora Integración Solar

El Grupo Breedon se ha asociado con el especialista en techado europeo Terran para lanzar Generon - un sistema de tejas solares discretas que combina:

- Celdas PV monocristalinas de 3.2mm
- Integrado con tejas base de concreto de 330mm x 420mm
- Instalación sin costuras con tejas de coincidencia estándar
- Sistema de 260 tejas genera una salida típica de 4kW

- Sujetadores de clip de tormenta estándar
