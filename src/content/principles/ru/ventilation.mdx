# Вентиляция с рекуперацией тепла: свежий воздух без потери энергии

Вентиляция с рекуперацией тепла (ВРТ) является важнейшим компонентом пассивных домов, обеспечивающим постоянную подачу свежего воздуха при сохранении энергоэффективности. Эта современная система вентиляции извлекает тепло из выходящего отработанного воздуха и использует его для нагрева поступающего свежего воздуха.

## Почему важна вентиляция с рекуперацией тепла?

В пассивном доме системы ВРТ выполняют несколько важных функций:

- **Энергоэффективность**: Возвращает до 90% тепла из отработанного воздуха
- **Качество воздуха**: Обеспечивает постоянную подачу свежего воздуха без открывания окон
- **Комфорт**: Поддерживает постоянный уровень температуры и влажности
- **Здоровье**: Фильтрует загрязнения, пыльцу и пыль
- **Контроль влажности**: Предотвращает конденсацию и рост плесени

## Как работает вентиляция с рекуперацией тепла?

Система ВРТ работает по простому, но эффективному принципу:

1. **Сбор отработанного воздуха**: Отработанный воздух извлекается из кухонь, ванных комнат и других влажных помещений
2. **Теплообмен**: Теплый отработанный воздух передает свое тепло поступающему свежему воздуху через теплообменник
3. **Распределение свежего воздуха**: Подогретый свежий воздух распределяется по жилым помещениям и спальням
4. **Непрерывная работа**: Система работает круглосуточно, обеспечивая постоянное качество воздуха

## Преимущества вентиляции с рекуперацией тепла

### Экономия энергии
- Возврат 80-90% тепла из отработанного воздуха
- Значительное снижение затрат на отопление
- Поддержание комфорта при минимальном потреблении энергии

### Улучшенное качество воздуха
- Постоянная подача фильтрованного свежего воздуха
- Удаление загрязнений из помещения
- Уменьшение количества аллергенов и пыли

### Комфорт и здоровье
- Отсутствие холодных сквозняков от открытых окон
- Равномерная температура по всему дому
- Снижение влажности и конденсации
